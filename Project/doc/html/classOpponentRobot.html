Detailed Description

Child class of NewRoboControl which desribes a robo of the other team.

This class does not implement anything and is just used to make the distinction with a TeamRobot. It is however still possible to use the driving functions of the parents NewRoboControl and "RoboControl", but this should be avoided as it is forbidden by the game rules.
