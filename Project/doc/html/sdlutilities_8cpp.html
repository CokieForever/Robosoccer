Sets the pixel value of the surface at the given point. The surface should be locked with SDL_LockSurface() before calling this function.

No check is done before trying to set the value: invalid coordinates can possibly lead to segfault or memory corruption. For a safer version, use PutPixelCheck().

Parameters:
surface The target surface
x x coordinate of the point
