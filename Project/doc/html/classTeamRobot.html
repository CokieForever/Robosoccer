Detailed Description

Abstract child class of NewRoboControl describing a robot from our team.

This class implements more high-level dring functions such as path computation / driving or kick functions. This class must be inherited to be constructed, and the child classes must implement the four virtual methods TeamRobot::setNextCmd(), TeamRobot::setCmdParam(), TeamRobot::performCmd() and TeamRobot::AddObstacleForFormation().
